Selected Projects
听见御花园 故宫合作 iOS
2025.09 — 2025.12
针对故宫北院展陈中环境缺失导致的体验断层，以声景视角整合御花园档案，串联「AIGC 声景 + 用户认知调研 + 空间音频动捕」链条，交付 Unity + Xcode iOS 移动端 2D 空间音频漫游游戏。

Citywalk 交互式 Web 应用
2025.09 — 2025.12
主导开发并上线一款交互式 Web 应用 — www.citywalkbeijing.com，整合小红书热点 POI 与地图落位，用户只需选定「出发起点、步行时长、兴趣类别」即可动态生成专属 Citywalk 路线。
数据构建：使用社媒助手爬取小红书北京热门 POI，调用高德地图 API 补全经纬度与大模型语义清洗，以极低成本完成空间数据库构建。约束导向交互机制：以步行时长为半径建立缓冲区约束，结合高德路径规划 API 生成最优串联路线，完成独立域名公网部署。
